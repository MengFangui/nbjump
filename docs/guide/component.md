# 组件

Mars 的组件规范使用百度智能小程序和微信小程序组件规范，在 H5 端进行了适配。

## 小程序组件组件文档

- [百度智能小程序](https://smartprogram.baidu.com/docs/develop/component/view/)
- [微信小程序](https://developers.weixin.qq.com/miniprogram/dev/component/)
